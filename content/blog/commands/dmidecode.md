## Introduction

DMIdecode is a tool to obtain Hardware informations.

## Mose useful commands

- System informations: `dmidecode -t1`
- Chassis informations: `dmidecode -t4`
- CPU informations: `dmidecode -t4`
- RAM informations: `dmidecode -t17`
